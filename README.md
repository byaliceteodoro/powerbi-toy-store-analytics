# 📊 Toy Store Analytics Dashboard

Dashboard interativo desenvolvido no Power BI para análise de desempenho de vendas de uma loja fictícia (toy store).

---

## 🎯 Objetivo
Analisar métricas de negócio e gerar insights sobre a performance de vendas, comportamento dos clientes e eficiência da conversão.

---

## 📌 Principais KPIs
- Receita Total  
- Total de Pedidos  
- Total de Sessões  
- Taxa de Conversão  

---

## 📈 Análises Realizadas
- Evolução da receita ao longo dos meses  
- Identificação de sazonalidade nas vendas  
- Relação entre volume de sessões e conversão  
- Monitoramento de desempenho comercial  

---

## 🛠️ Ferramentas Utilizadas
- Power BI  
- Power Query  
- Modelagem de Dados

## 💡 Principais Insights
- Crescimento de receita no último trimestre do ano  
- Queda de desempenho no período intermediário (abril a junho)  
- Relação direta entre aumento de sessões e crescimento de pedidos  
- Taxa de conversão estável com oportunidades de otimização

## 🚀 Possíveis Melhorias
- Análise detalhada por produto  
- Segmentação de clientes (novos vs recorrentes)  
- Funil de vendas completo  
- Integração com Python para análises mais avançadas
